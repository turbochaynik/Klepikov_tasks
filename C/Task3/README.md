application.h
Заголовочный файл, содержащий описание типа Array и объявления функций, определенных в application.c.

Array представляет собой структуру со следующими полями:
 • char **word - массив строк (слов).
 • size_t size - количество строк в массиве.
 • size_t capacity - текущий объем памяти, выделенный под массив.

application.c
Этот файл содержит реализации функций для работы с типом Array.

*void sort(Array list)
Принимает указатель на тип Array и сортирует массив строк в структуре list с использованием метода пузырьковой сортировки.

**void add(Array list, char word)
Принимает list - указатель на Array и строку word. Добавляет строку word в конец массива строк list. Если выделенной памяти недостаточно для нового элемента, память перевыделяется с удвоением текущего объема.

*void free_list(Array list)
Освобождает память, выделенную для каждого элемента массива строк, а затем освобождает память, выделенную для самого массива строк.

*void print_list(Array list)
Выводит все элементы массива строк в порядке их расположения.

main.c
Реализует чтение слов из stdin и вывод слов в stdout.

Buf - буфер, представляющий собой структуру с полями:
 • char *data - строка для хранения промежуточного слова.
 • size_t size - текущая длина слова.
 • size_t capacity - текущий объем памяти, выделенной под слово.

Str - структура, используемая для хранения данных, считанных из stdin, с последующим посимвольным чтением.

Функция getsym считывает следующий символ из строки str. Если текущий индекс достиг конца данных в str->data, происходит новое чтение из stdin с использованием fgets, а затем возвращается следующий символ.

Функция add_to_buf добавляет символ c в конец строки buf->data. Если выделенной памяти недостаточно, она увеличивает емкость buf->data на SIZE и перевыделяет память с помощью realloc.
